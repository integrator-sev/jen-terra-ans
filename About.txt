  Jenkins генерирует пару ключей, создает пару машин в AWS с помощью Terraform.
  Jenkins конфигурирует созданные машины с помощью Ansible.
  Jenkins клонирует репозиторий на машину сконфигурированную для сборки приложения, собирает Docker Image с приложением и отправляет его в Docker Hub.
  Jenkins на машину сконфигурированную для запуска приложения копирует собранный Docker Image из Docker Hub и запускает контейнер с приложением.

P.S. Для работы необходимо наличие действующего аккаунта AWS и использование плагина авторизации Jenkins.
